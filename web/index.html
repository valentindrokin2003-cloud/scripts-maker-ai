Сборка сценария

Сервис собирает аналитический notebook по брифу без ручной подготовки.

Загрузите Excel-файл клиента, а сервис сам проведет разбор брифа, подготовит параметры выборки и вернет готовый рабочий notebook для команды аналитики.

Входной документ: Excel-бриф клиента
Извлечение: LLM + правила сервиса
Результат: Очередь + `.ipynb` архив
